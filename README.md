This is a [Next.js](https://nextjs.org/) combine with [GraphQL](https://graphql.org/) project for authentication using JWT

## Technical Stack

Front-end / Client:

```
- NextJS: React Framework
- urql: GraphQL Client
- TailwindCSS: CSS Framework
```

Back-end / Server:

```
- Express
- GraphQL API
- Nexus: Code-First GraphQL Schemas
- Prisma: ORM
```
